# Alpha Relay Topology

Alpha Relay is a source node in the synthetic associative graph benchmark.
Alpha Relay connects to Bridge Junction through the Alpha Bridge Link.
Bridge Junction routes to Gamma Endpoint through the Bridge Gamma Route.
Gamma Endpoint stores the final delivery state for the benchmark route.

Ordinary Artifact is also mentioned by Alpha Relay, but Ordinary Artifact is not part of the route to Gamma Endpoint.
The route from Alpha Relay to Gamma Endpoint is indirect and must pass through Bridge Junction.
